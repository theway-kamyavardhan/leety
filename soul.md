# soul.md - Placement Prep Memory

This file is the portable memory format for the Aptitude + IELTS module.

The web app stores live test history in browser `localStorage`. Because a deployed browser app cannot silently write back into the GitHub repo, use the **soul.md** button inside the Aptitude + IELTS tab after major tests to download an updated copy.

## What It Tracks

- Quant, Verbal, Reading Comprehension, and IELTS test scores
- Weak topics and accuracy
- Adaptive 7-day plan
- DSA/job/mock snapshot from the tracker
- Next study actions

## Suggested Workflow

1. Take the initial diagnostic.
2. Download `soul.md`.
3. Keep the downloaded file as your portable reference.
4. Retake daily adaptive practice.
5. Download a fresh `soul.md` after each major test or weekly review.
